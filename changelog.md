# Changelog

## v1.9.2

### Added
- Expanded monitored inventory list to include store-bought provisions (carrots, celery, peaches, cherries, pears).

### Changed
- Refined Auto-Aim lock target to focus on the chest/torso instead of the head to ensure bullet hits.
- Changed Loot Announcer prefix from "Looted" to "Added" for neutral store purchase and looting notifications.
- Fixed rapid speech clipping in Loot Announcer by concatenating overlapping speech events.
- Renamed the trainer build to "RDR access.asi" and updated all packaging.

### Removed
- Completely removed Navigation Mode and related menus/hotkeys.

## v0.1

### Added

- Initial release of the RDR2 Accessibility Mod.
- Added accessibility navigation mode with spoken POI and mission browsing.
- Added mission/stranger/activity browser using Numpad 4 and Numpad 6.
- Added auto-navigation with Numpad 8 and stop toggle support.
- Added auto-walk and auto-run commands on the nav mesh.
- Added horse speed settings for mounted navigation.
- Added bodyguard squad manager.
- Added human and animal bodyguard spawning.
- Added guard horse support with faster mounted follow behavior.
- Added Numpad 1 guard status announcement with alive/fighting counts and type breakdown.
- Added automatic guard mount/dismount synchronization with player mount state.
- Added auto-aim support to assist aiming for blind players.
- Added aiming announcements for enemies, civilians, animals, vehicles, objects, and empty aim.
- Added horse whistle/call detection and arrival announcement.
- Added accessibility auto-announcements for money, horse state, guard status, compass, and animal behavior.
